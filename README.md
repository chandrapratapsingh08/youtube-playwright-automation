# youtube-playwright-automation
Playwright automation to play the latest Dhruv Rathee video on YouTube

## Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- - npm or yarn
 
  - ### Steps to Run Locally on MacBook
 
  - 1. Clone the repository:
    2. git clone https://github.com/chandrapratapsingh08/youtube-playwright-automation.git
    3. cd youtube-playwright-automation
   
    4. 2. Install dependencies:
       3. npm install playwright
      
       4. 3. Run the automation script:
          4. node play-dhruv-rathee.js
         
          5. ### What the script does:
          6. - Launches a Chromium browser
             - - Navigates to Dhruv Rathee's YouTube channel
               - - Automatically plays the latest video
                 - - Displays a success message in the console
                  
                   - ### Requirements
                   - - Internet connection to access YouTube
                     - - Playwright (installed via npm)
                       - - Node.js installed on your MacBook
                        
                         - ### Notes
                         - - The browser window will open and close automatically
                           - - Check the console output for success/error messages
